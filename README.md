# Usage

link your `src` directory.
`src` structure:
```
src
├── 0 Data Structure
│   ├── CartesianTree.cpp
│   ├── ...
│   ├── TreapRotate.cpp
│   └── pb_ds.cpp
├── 1 Dynamic Programming
│   ├── PlugDP-circuits.cpp
│   ├── ...
│   └── SlopeDP.cpp
├── 2 FFT
│   ├── FFT.cpp
│   ├── ...
│   └── SelfConvolution.cpp
├── ...
├── 9 Notes
│   ├── Conclusion.tex
│   ├── ...
│   └── Graph.tex
...
```
Run "make" in shell and have fun.
Use "<TeX>" "</TeX>" in the beginning of your code to escape TeX.
